## Project Overview
This project analyzes bike sales data using an Excel dashboard to understand customer behavior and sales performance.

## Tools Used
- Microsoft Excel

## Key KPIs
- Total Sales
- Total Customers
- Average Order Value
- Revenue by Age Group
- Commute Distance Analysis

## Insights
- Middle-aged customers generate higher sales
- Certain commute distances show stronger purchase trends
- Customer segmentation helps identify target groups
